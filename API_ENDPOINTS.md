# API Endpoints

## Public

- `GET /api/public/video-popup`
- `POST /api/events/product-view`
- `POST /api/events/search`
- `POST /api/events/cart`

## Admin Analytics

- `GET /api/admin/analytics/customer-behavior`
- `GET /api/admin/analytics/product-interest`
- `GET /api/admin/analytics/abandoned-carts`
- `GET /api/admin/analytics/search-misses`

## Sales

- `POST /api/admin/sales-opportunities/create`
- `POST /api/admin/sales-tasks/create`

## Import

- `GET /api/admin/import/sources`
- `POST /api/admin/import/xml`
- `POST /api/admin/import/pdf`
- `POST /api/admin/import/preview`
- `POST /api/admin/import/approve`
- `POST /api/admin/import/reject`
- `POST /api/admin/import/export-xml`

## Settings

- `GET /api/admin/settings/video-popup`
- `POST /api/admin/settings/video-popup`

Admin endpointleri `entas_admin_session` cookie ister.
